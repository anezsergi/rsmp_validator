# forcing is set with M0019

1. Verify connection
2. Send control command to set force input
3. Wait for status = true

```ruby
Validator::Site.connected do |task,supervisor,site|
  status = 'False'
  input = 1
  inputValue = 'True'
  prepare task, site
  force_input status, input, inputValue
end
```
