# dynamic bands is set with M0014

1. Verify connection
2. Send control command to set dynamic_bands
3. Wait for status = true

```ruby
Validator::Site.connected do |task,supervisor,site|
  plan = "1"
  status = "10,10"
  prepare task, site
  set_dynamic_bands status, plan
end
```
