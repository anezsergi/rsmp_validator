# day table is set with M0017

1. Verify connection
2. Send control command to set time_table
3. Wait for status = true

```ruby
Validator::Site.connected do |task,supervisor,site|
  status = "12-1-12-59,1-0-23-12"
  prepare task, site
  set_time_table status
end
```
