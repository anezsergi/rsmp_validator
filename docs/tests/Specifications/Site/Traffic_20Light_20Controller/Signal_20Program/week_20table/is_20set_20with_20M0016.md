# week table is set with M0016

1. Verify connection
2. Send control command to set  week_table
3. Wait for status = true

```ruby
Validator::Site.connected do |task,supervisor,site|
  status = "0-1,6-2"
  prepare task, site
  set_week_table status
end
```
