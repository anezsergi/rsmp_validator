# cycle time is set with M0018

1. Verify connection
2. Send control command to set cycle time
3. Wait for status = true

```ruby
Validator::Site.connected do |task,supervisor,site|
  status = 5
  plan = 0
  prepare task, site
  set_cycle_time status, plan
end
```
